# Jewish Bible Calendar v7 - Meditation Notes

추가 기능:
- 오늘의 묵상 노트 카드
- 날짜별 메모 자동 저장
- 저장 / 복사 / 삭제 버튼
- 한국어/영어 placeholder
- 메모는 서버로 전송되지 않고 사용자의 브라우저 localStorage에만 저장됨

주의:
- 휴대폰/브라우저를 바꾸면 메모가 옮겨지지 않습니다.
- 브라우저 데이터 삭제 시 메모도 삭제될 수 있습니다.
